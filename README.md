# tensorflow_loldata
2025 텐서플로우 롤 데이터 분석과제
조원: 우정광, 김준성, 이장헌

초기에 토의를 통해 결정한 지표들이다.
**모든 데이터는 14분 이전의 분당 데이터이다 (모든 데이터를 끌고오되 결과를 보고 14분으로 끊을 예정).**
| 탑 |                                                                   |
|:----|:--------------------------------------------------------------------|
| **cs(t)** | 미니언 처치 횟수로, 직접적으로 골드에 직결되는 지표    |
| **경험치(t)** |   라인을 완전히 버리지 않은 채로 진행하였는가를 판단할 지표  |
| **현재 체력 차이(t)** |     상대와의 현재 체력 차이로, 딜교환을 얼마나 진행하였는지를 엿볼 수 있다.      |
| **현재 입힌 피해량(t)** |      입힌 모든 피해    |
| **챔피언 외 피해량(t)** |        라인과 타워에 입힌피해량으로 피해량의 차이로 누가 더 라인을 주도하고 있는지를 알아낼 지표      |
| **KDA** |  킬 뎃 어시스트의 누적으로 골드에 직결되는 지표  |



| 미드 |                                                                   |
|:----|:--------------------------------------------------------------------|
| **cs(t)** | 분 단위의 미니언 처치 횟수                                |
| **경험치(t)** |   라인을 완전히 버리지 않은 채로 진행하였는가를 판단할 지표  |
| **현재 체력 차이(t)** |     상대와의 현재 체력 차이로, 딜교환을 얼마나 진행하였는지를 엿볼 수 있다.      |
| **현재 입힌 피해량(t)** |      입힌 모든 피해    |
| **챔피언 외 피해량(t)** |        라인과 타워에 입힌피해량으로 피해량의 차이로 누가 더 라인을 주도하고 있는지를 알아낼 지표      |
| **KDA** |  킬 뎃 어시스트의 누적으로 골드에 직결되는 지표  |




| 정글 |                                                                   |
|:----|:--------------------------------------------------------------------|
| **cs(t)** | 분 단위의 정글 캠프 처치 횟수 |
| **경험치(t)** |  상대와의 경험치 차이를 통해 상대와의 동선 비교  |
| **KDA** |  킬 뎃 어시스트의 누적으로 골드에 직결되는 지표  |
| **오브젝트 처치** |   오브젝트 처치 관여  |
| **시야점수** |   와드를 설치하고 파괴한 횟수  |




| 바텀 |                                                                   |
|:----|:--------------------------------------------------------------------|
| **cs(t)** | 분 단위의 미니언 처치 횟수(원딜)                                |
| **경험치(t)** |   라인을 완전히 버리지 않은 채로 진행하였는가를 판단할 지표(둘 다)  |
| **현재 체력 차이(t)** |     상대와의 현재 체력 차이로, 딜교환을 얼마나 진행하였는지를 엿볼 수 있다.(둘 다)      |
| **현재 입힌 피해량(t)** |      입힌 모든 피해(둘 다)    |
| **챔피언 외 피해량(t)** |        라인과 타워에 입힌피해량으로 피해량의 차이로 누가 더 라인을 주도하고 있는지를 알아낼 지표(둘 다)      |
| **KDA** |  킬 뎃 어시스트의 누적으로 골드에 직결되는 지표  (둘 다) |
| **시야점수** | 와드를 설치하고 파괴한 횟수 |
